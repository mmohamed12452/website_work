>PUBLISHING - It does not seem to recognise public_html and places them in root directory

Did you specify the public_html folder in the Publish window?

>Images with Flip Animations - how to change TEXT on flip?

Click on the bottom left corner of the card to change the front image. In browser this corner will not be shown.

> How to specify more text styles in addition to: Title1/Title2/Title3/Text/Menu.

Unfortunately, it isn't supported. But you can change your test style manually using Code Editor.

> How to insert a SMALL slideshow within a block...with transition options/time options.

You can't simply add specific elements to your blocks. Please use a block that already includes a slider.

> Used my own image on Header block (1920 x 1080) but part of image is not showing on mobile view.

Mobirise sites are responsive, and blocks change their size, so it's impossible to keep the same size of images. We preserve aspect ratio, but images can look cropped.

> I like to put a different (bigger) menu on mobile versions and also place a 'Click to Call' button on it (pointless on desktop).

Please use the Code Editor extension to make changes in code of your blocks, it isn't supported.
